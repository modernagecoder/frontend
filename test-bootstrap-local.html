Then visit: http://localhost:8000/src/pages/bootstrap.html

1. Install "Live Server" extension in VS Code

2. Right-click on src/pages/bootstrap.html

üîç Troubleshooting

Issue: /bootstrap shows 404

Solution:

  • Make sure you're using netlify dev (not python or http-server)
  • Check that _redirects file exists in root
  • Verify the line: /bootstrap /src/pages/bootstrap.html 200 is in _redirects
  • Restart the server

Issue: Components not loading

Solution:

  • Check browser console for errors
  • Verify /components/nav.html exists
  • Verify /components/footer.html exists
  • Verify /src/js/components-loader.js exists

Issue: Styles not loading

Solution:

  • Check that /src/css/style.css exists
  • Check that /src/css/responsive.css exists
  • Check browser console for 404 errors
  • Verify paths in HTML are correct
