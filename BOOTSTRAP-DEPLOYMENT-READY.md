## ðŸ”§ Troubleshooting

### Issue: Page shows 404
**Solution:**
1. Check `_redirects` file has the route
2. Check `netlify.toml` has the redirect
3. Clear browser cache
4. Redeploy site

### Issue: Navigation/Footer not loading
**Solution:**
1. Check `/src/js/components-loader.js` exists
2. Check `/components/nav.html` exists
3. Check `/components/footer.html` exists
4. Check browser console for errors
5. Verify paths are correct

### Issue: SEO tags not showing
**Solution:**
1. View page source (Ctrl+U)
2. Check `<head>` section
3. Verify meta tags are present
4. Clear CDN cache if using one

### Issue: Analytics not tracking
**Solution:**
1. Check Google Analytics code is present
2. Verify `gtag` function exists
3. Check browser console for errors
4. Test in incognito mode
5. Wait 24-48 hours for data
