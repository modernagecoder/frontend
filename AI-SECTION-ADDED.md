## üé® Customization

### Change Colors
Edit in CSS:
```css
.bs-ai-card {
    border: 1px solid rgba(168, 85, 247, 0.2);
}
```

### Change Grid Columns
Edit in CSS:
```css
.bs-ai-grid {
    grid-template-columns: repeat(3, 1fr);
}
```

### Add More Cards
Copy any `.bs-ai-card` block in HTML and modify content.

### Change Icons
Replace Font Awesome icon classes:
```html
<i class="fas fa-robot"></i>
```
