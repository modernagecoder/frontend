- [ ] **H1**: One per page, includes primary keyword

- [ ] **Image alt tags**: Descriptive, include keyword where natural

- [ ] **Page speed**: All pages < 3 seconds load time

- [ ] **Open Graph tags**: Title, description, image for every page
- [ ] **Twitter cards**: Set up for social sharing
- [ ] **Breadcrumbs**: Add BreadcrumbList schema to all pages
