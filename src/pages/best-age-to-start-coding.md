# What is the best age to start coding? Younger than you fear, later than you think

## The age map

4-5: not formal classes — puzzle play and pattern games; we say "wait" and mean it. 5-6: borderline, picture blocks IF signals are strong. 6-9: the sweet spot — Scratch taught for ideas. 9-12: either door — fast-tracked Scratch or straight Python; the single best range for a text start. 13-16: Python directly, real projects fast (blocks feel babyish, rightly). 17+: start anywhere; motivation replaces the head start.

## The six readiness signals (worth more than age)

Asks how things work · builds and tinkers (LEGO/Minecraft) · tolerates not-working-yet · reads fluently enough for the tool · holds a 20-minute focus on chosen things · wants it (or would, if shown).

## The two fears

Too-early: partly right, badly aimed — drills and syntax at 6 backfire; play-shaped teaching flourishes. Too-late: almost entirely wrong — there is no closing window; motivated teens catch casual early starters within months.

## What early actually buys

Runway, not superiority: pressure-free years to wander into game design, AI or contests before any exam cares. The late start buys focus. Both roads reach fluency.
