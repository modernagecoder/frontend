## Understand &amp; Build with LLMs

From transformers to shipping a real LLM-powered app.

### How LLMs Work

Tokens, embeddings, attention and transformers — a clear mental model of what GPT-style models really do.

### Build with LLM APIs

Call LLM APIs from Python, structure prompts, and wire models into real applications.

### RAG &amp; Fine-Tuning

Ground LLMs in your own data with retrieval-augmented generation, and understand when and how fine-tuning helps.

## The LLM Roadmap

**STAGE 01 — Transformers**: Tokens, embeddings and attention — how LLMs represent and process language.

**STAGE 02 — Build with APIs**: Use LLM APIs in Python to generate, summarise and reason.

**STAGE 03 — RAG**: Add retrieval so the model answers from your own documents.

**STAGE 04 — App &amp; Fine-Tune**: Ship an LLM-powered app and explore fine-tuning.
