# Scratch coding classes for kids: real programming, disguised as play

**The 20-second answer:** teach the ideas inside the blocks (not recipes), let the child build their own projects, and graduate them to Python when ready.

## The decoder

Ages 5-6: ScratchJr-style picture blocks. Ages 6-9: Scratch sweet spot. Ages 10-11: Scratch fast-track with Python in sight. Ages 12+: usually straight to Python. Free demo = placement with evidence.

## Signature teaching

The cat-and-mouse chase game: nine blocks carrying four foundation ideas (event, forever loop, conditional, score variable) in lesson three; the mentor's questions ("what if 5 becomes 50?") move ideas from screen into child. Month 10+: the Scratch-to-Python bridge made visible, repeat(100)/move/if-touching-edge re-met as a Python for-loop, so the child says "that is just my blocks written in words."

## The path

Demo → foundation four (events/loops/conditionals/coordinates) → game craft (variables, cloning, broadcasting) → maker phase (self-chosen multi-level builds, shipped to Student Labs) → the Python bridge with the same mentor.
