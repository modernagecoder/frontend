# How much do coding classes for kids cost in 2026?

## Costs by format (typical published US prices, July 2026)

| Format | Typical cost |
|---|---|
| Self-paced apps (Tynker) | ~$180/year, no live teacher |
| Class marketplaces (Outschool) | $10 to $30 per class hour; series/1-on-1 $50 to $100+ |
| Semi-private programs (Juni) | from ~$160/month, 2 students |

| Live online schools (Modern Age Coders) | $40 group to $100 1-on-1 per month, 8 live one-hour classes |

Sources: myelearningworld.com, brighterly.com, tynker.com, masandpas.com, provider sites.

## What drives the price

Teacher or no teacher; where the teachers live (US wages vs global); frequency (4 vs 8 classes a month); structure vs sampling; class size; what is bundled (maths, project reviews, recordings, mentor continuity).
