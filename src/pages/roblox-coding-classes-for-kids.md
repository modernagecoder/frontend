# Roblox coding classes for kids: from playing games to building them

**The 20-second answer:** real Luau programming in Roblox Studio (variables, events, functions, loops), not terrain painting and free models; the obsession redirected into a skill; ages 9-14 sweet spot.

## Signature teaching

Week two: the touch-activated door — seven lines of Luau carrying a variable, a self-defined function, property changes and an event connection; the child walks their avatar into a solid door and it turns to glass because THEIR code said so. Months 4-6: the leaderboard (PlayerAdded, Instance.new, leaderstats) — services, runtime object creation and state, arrived at because their tycoon needed coins.

## Safety, honestly

Classes happen in Roblox Studio, the development tool: no public servers, no strangers. We brief parents once on the public platform's account controls, chat settings and spending locks. Economy taught as literacy with honest odds: a handful of teens earn meaningfully; the guaranteed payoff is the programming.

## The path

Demo (scripts something real in hour one) → Studio fluency + first scripts → first published game (obby/adventure with checkpoints and win states) → systems (leaderboards, currencies, saving data) → the builder's fork: deeper Roblox or the Python bridge, same mentor.
