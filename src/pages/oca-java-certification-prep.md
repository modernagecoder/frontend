## What OCA Java SE 8 is
OCA — Oracle Certified Associate, Java SE 8 Programmer is earned by passing one exam: Java SE 8 Programmer I (1Z0-808), issued by Oracle, the steward of Java. It validates that you can read and write correct Java SE 8 code across the language core: data types, operators, control flow, arrays, methods, object-oriented programming, exceptions and core API classes.

**Accuracy note:** most blogs still print 70–77 questions / 150 minutes — that is the obsolete 2014 launch spec. Oracle's live exam page lists **56 questions, 120 minutes, 65% to pass**. Use the current numbers.

## Exam at a glance (1Z0-808)
- Questions: 56 (multiple choice) · Duration: 120 minutes · Passing score: 65% (~37 of 56)
- Price: about US$245 (Oracle says it varies by country/currency — confirm at checkout)
- Delivery: Pearson VUE (test center or online proctored)

## Full exam objectives (9 areas)
1. **Java Basics:** variable scope; class structure; `main` method and running from the command line; importing packages; platform independence, OOP, encapsulation.
2. **Working with Java Data Types:** declare/initialise/cast primitives; object references vs primitives; object lifecycle and garbage collection; wrapper classes (`Boolean`, `Integer`, `Double`).
3. **Operators & Decision Constructs:** operators and precedence; `==` vs `equals()`; `if`/`if-else`/ternary; `switch`.
4. **Creating & Using Arrays:** one-dimensional and multi-dimensional arrays.
5. **Loop Constructs:** `while`, `for`, enhanced `for`, `do/while`; `break`, `continue`.
6. **Methods & Encapsulation:** methods with args/returns; overloading; `static`; constructors (default vs user-defined); access modifiers; encapsulation; pass-by-value effects.
7. **Inheritance:** inheritance, polymorphism, overriding; reference type vs object type; casting; `super`/`this`; abstract classes and interfaces.
8. **Handling Exceptions:** checked vs unchecked vs Errors; `try-catch` and flow; throwing exceptions; common classes (`NullPointerException`, `ArithmeticException`, `ArrayIndexOutOfBoundsException`, `ClassCastException`).
9. **Selected Java API Classes:** `String`, `StringBuilder`; `java.time` (`LocalDate`, `LocalDateTime`, `Period`, `DateTimeFormatter`); `ArrayList`; a simple Lambda with a `Predicate`.

## OCA SE 8 vs the modern Java SE 17 (OCP)
- **OCA Java SE 8 (1Z0-808):** legacy foundational cert, still active. 56 Q / 120 min / 65%. Two-step SE 8 path: OCA (1Z0-808) → OCP (1Z0-809).
- **Java SE 17 Developer (1Z0-829):** Oracle's current credential. 50 Q / 90 min / 68%. From Java SE 11 onward Oracle removed the separate associate tier — a single exam earns the full professional OCP. Broader scope: streams, lambdas, modules (JPMS), concurrency, JDBC, records, sealed classes.

Our take: OCA SE 8 is an approachable, recognised first Java certificate; SE 17 is the current professional credential. We coach both.

## How to prepare
- Complete beginner: 8–12 weeks. Comfortable with basic Java: 4–6 weeks. Experienced developer: 2–3 weeks (mostly the exam's edge cases).
- Study the nine objectives in order, compile and run snippets to confirm real behaviour, and drill timed mock exams until comfortably above 65%.

## Common pitfalls (FAQ fodder)
String immutability and `==` vs `equals()`; autoboxing and the Integer cache (−128 to 127); variable scope and default values (locals have none); overloading vs overriding; operator precedence and integer/char arithmetic; `switch` fall-through; short-circuit `&&`/`||`; array vs ArrayList; exception hierarchy and `finally`; `this`/`super` and constructor chaining; abstract classes vs interfaces; pass-by-value semantics.
