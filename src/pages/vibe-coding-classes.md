## How a vibe coding class runs

1. **Build first** — each class opens with a real goal, so students write and run code from the start.
2. **Direct the AI** — learn to brief AI tools well, then read and improve what they produce (age-appropriate: friendly kids' tools up to Cursor, Lovable and Claude).
3. **Understand it** — the mentor unpacks the fundamentals, logic and maths behind what was built.
4. **Ship & review** — finish with something that works, with live code review in a 4-8 student group and a recording to revise.

## Frequently Asked Questions

**What is vibe coding?**
Vibe coding is building real software — apps, games and AI projects — by describing what you want and directing AI tools to help build it, while still learning the programming underneath. Students use AI alongside genuine coding fundamentals, so they become makers who can read, change and explain their own code.

**Which vibe coding course is right for my age?**
Kids (8–12): Scratch, game dev and AI creative projects. Teens (13–17): Python, web development and AI tools like Cursor, Lovable and Claude. College (18+): MERN full-stack, DSA, Machine Learning and system design for placements.

**Is vibe coding just letting AI write the code?**
No. AI is a tool we teach students to direct, not a replacement for understanding. Learners build real projects with AI and learn the fundamentals underneath — how the code works, why it works, and how to fix it.

**Does my child need prior coding experience?**
No prior experience is needed for the Kids and Teens tracks. The College track expects basic programming familiarity. During enrolment we help each student pick the right track.

**How long are the courses?**
Each track runs 3 months (12 weeks) with two live classes per week: about 4–6 hours/week for Kids, 6–8 for Teens, 10–15 for College.

**What AI tools are used?**
Age-appropriately — friendly safe tools for kids; Cursor IDE, Lovable and Claude for teens; AI assistants and agents for college. Dedicated Codex and Claude Code masterclasses are available for teens and adults.

**Are classes live or recorded?**
Live and interactive in small batches of 4-8, never pre-recorded. Every session is recorded for revision and mentors review code in real time.

**How much do vibe coding classes cost?**
Group ₹1,499/month, Mini-Batch ₹2,499/month, 1-on-1 ₹4,999/month. Lifetime access available per track. Free trial included; international pricing shown automatically outside India.

**Do students get a certificate?**
Yes — a completion certificate and a real portfolio. For college students, the portfolio and capstone support placements and internships.

**Is vibe coding a good way to learn to code in 2026?**
Yes, when taught properly. Directing AI is now part of how software is built, but the engineers who benefit most understand code. We teach both at once.
