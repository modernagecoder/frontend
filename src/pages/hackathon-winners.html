Deepfake Detection

Catching AI-generated fake videos with 93.59% accuracy — frame by frame.

₹6,000Rank 01 · Top Prize

As AI-generated video gets harder to spot, this project fights back with deep learning. It pairs a ResNext50 CNN — which reads the fine facial and skin artefacts inside each frame — with an LSTM that learns the unnatural patterns deepfakes leave across frames over time. Trained on 6,000 videos from FaceForensics++, Celeb-DF and the Deepfake Detection Challenge, it's wrapped in a clean Django web app: upload a clip and get a real-or-fake verdict with a confidence score. The whole thing is Dockerised to run on either GPU or CPU.

93.59%Accuracy
6,000Videos Trained
CNN + LSTMHybrid Model

Key Features

  • Hybrid ResNext50 + LSTM architecture using transfer learning
  • Accuracy scales from 84% (10 frames) to 93.59% (100 frames)
  • Upload-and-detect Django app with live confidence score
  • Automatic face detection & cropping (face_recognition / dlib)
  • Docker + NGINX deployment, GPU and CPU support

How It Works

Video → frames → face crop feed each face into ResNext50 to extract spatial features, pass the sequence through the LSTM to model time-based patterns, then output Real / Fake + confidence %.

AgencyOS

An AI operating system that runs an entire agency from plain-English commands.

₹5,000Rank 02

AgencyOS is a full-stack SaaS platform that systematises the eight core layers of running an agency — Strategy, People, Sales, Delivery, Finance, Knowledge, Client Success and Reporting — into one control plane. Instead of clicking through forms, you type what you want in plain English. A lightweight TypeScript ReAct orchestrator called Hermes reasons over an MCP toolset and 12+ integrations to actually get it done. It's built privacy-first: bring your own AI key, every action is audit-logged, and the whole stack can be self-hosted into your own cloud with a single command.

Blue Carbon Credit Registry

A blockchain registry that turns mangrove restoration into verified, tradable carbon credits.

₹4,000Rank 03

This platform builds a transparent marketplace for blue carbon credits — credits earned by restoring coastal and ocean ecosystems like mangroves. It connects three roles end to end: NGOs submit restoration projects with tree counts, GPS coordinates and photos; auditors verify them through a rigorous 10-stage workflow; and buyers purchase the resulting credits. Every verified project is written to a blockchain ledger as an immutable, SHA-256 hash-linked block, and tokens are minted to the NGO — so buyers can see exactly which auditor approved a project and how trustworthy that auditor's track record is.

3 PortalsNGO · Auditor · Buyer
10-StageVerification
SHA-256Hash-Chain Ledger

Key Features

  • Role-based portals for NGOs, auditors and customers
  • 10-stage verification pipeline with per-auditor trust scores
  • Blockchain ledger: genesis block + SHA-256 hash chain
  • Token minting (1 token per tree) with a full transfer ledger
  • Transparent buyer view of verified projects + auditor accuracy
  • Real-time updates, notifications, accessible & fully responsive

How It Works

NGO submits project → auditor approves 10 stages a new block is added to the chain and tokens are minted to the NGO, then buyers browse the verified chain and purchase credits.
