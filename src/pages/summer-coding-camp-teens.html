Common Questions

Frequently Asked Questions

Do I need prior coding experience?

No experience required! We start from the basics. If you've never written a line of code, you're in the right place. Prior exposure helps but isn't necessary.

What equipment do I need?

A laptop or desktop (Windows/Mac/Linux), stable internet connection, and headphones with microphone. All software we use is free.

How long is each session?

Each session is 90 minutes - long enough for meaningful learning but short enough to maintain focus. Sessions include hands-on coding practice.

Will I get homework?

Yes, but fun homework! You'll get small coding challenges and project extensions to practice between sessions. Expect 30-60 minutes of practice daily.

Can I use these skills for college/career?

Absolutely! The skills (HTML, CSS, JavaScript, Python) are the same ones used by professional developers. Your portfolio can be linked in college applications and resumes.

What if I miss a class?

All sessions are recorded. You can watch the recording and schedule a 15-minute doubt-clearing call with the instructor if needed.
