## What PCEP is
PCEP — Certified Entry-Level Python Programmer is the entry-level certification from the OpenEDG Python Institute. It proves you understand universal programming concepts and the core of Python: data types, operators, conditionals, loops, the four built-in collections (lists, tuples, dictionaries, strings), functions and exception handling. It is vendor-neutral, recognised worldwide, has no prerequisites, and is the official interim step toward the associate-level PCAP.

**Version note:** the current exam is PCEP-30-02 (lifetime certificate), scheduled to retire 31 August 2026; the successor PCEP-30-03 moves to an 8-year renewable validity. Always confirm the live version on the Python Institute site before booking.

## Exam at a glance (PCEP-30-02)
- Questions: 30 (single/multi-select, drag & drop, gap fill, code insertion)
- Duration: 40 minutes (+ ~5 min NDA/tutorial; some sources say 45 min)
- Passing score: 70% cumulative across the four blocks
- Price: from about USD 69 (bundles available); taken online via OpenEDG TestNow
- Proctoring: non-proctored, open-book
- Prerequisites: none · Validity: lifetime (PCEP-30-02) · Retake: 7-day wait after a fail

## Full syllabus (4 blocks)
1. **Computer Programming & Python Fundamentals — 18%:** interpreting vs compiling; keywords, indentation, comments, PEP-8; literals and numeral systems (binary/octal/decimal/hex); numeric, string, assignment, unary/binary, bitwise, Boolean and relational operators; `print()`, `input()`, `sep=`/`end=`, type casting.
2. **Control Flow: Conditions & Loops — 29%:** `if`/`elif`/`else`, nested conditions; `while` and `for`, `range()`, `in`; `break`, `continue`, `pass`; `while-else` and `for-else` (the `else` runs only if no `break`).
3. **Data Collections — 25%:** lists (indexing, slicing, methods, comprehensions, copying vs cloning, matrices); tuples (immutability); dictionaries (keys/values/items, iteration); strings (slicing, immutability, escaping).
4. **Functions & Exceptions — 28%:** defining/calling functions, `return`, `None`, recursion, generators; parameters vs arguments, positional/keyword/default, scope, shadowing, `global`; the exception hierarchy (`BaseException`, `Exception`, `ArithmeticError`, `LookupError`, `IndexError`, `KeyError`, `TypeError`, `ValueError`); `try-except`, ordering except branches, propagation.

Control flow (29%) and Functions & Exceptions (28%) together are ~57% of the exam.

## Certification path
PCEP (entry) → PCAP (associate, adds OOP) → PCPP1 → PCPP2 (professional). PCEP is positioned as the interim step toward PCAP.

## How to prepare
Beginner timeline: ~6–8 weeks at 5–7 hrs/week, or 4–6 weeks at ~10 hrs/week. Study in weighted order (fundamentals → control flow → collections → functions & exceptions), code from week one, and finish with timed practice on the real interactive question formats.

## Common pitfalls (FAQ fodder)
Operator precedence and binding; `/` (float) vs `//` (floor) division incl. negatives; modulo with negatives; slicing edge cases; mutability of list vs tuple vs string; copying vs cloning lists; `print()` `sep=`/`end=`; logical vs bitwise operators; variable scope and shadowing; function argument types; exception hierarchy and `except` ordering; `for-else`/`while-else`.
