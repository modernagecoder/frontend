Modern Age Codersthe toolkit
twelve methods, the whole syllabus
length()
How many characters
charAt(i)
The character at index i
indexOf(x)
Where x first appears, or -1
substring(a, b)
Slice from a to b-1
replace(old, new)
Swap every occurrence
trim()
Strip leading and trailing spaces
equals(s)
Same characters, case matters
equalsIgnoreCase(s)
Same characters, any case
compareTo(s)
Alphabetical order as a signed int
toUpperCase() / toLowerCase()
A new string in the new case
startsWith(s) / endsWith(s)
Boundary checks, true or false
split(" ")
Break a sentence into words
positions? charAt + substring. comparing? equals + compareTo.
