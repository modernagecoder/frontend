Modern Age Codersfunctions or classes?
pick the tool that fits the job
Plain functions win when...
  • the script is short and runs once
  • data flows in, an answer flows out
  • there is only ever one of everything
Classes win when...
  • data and behaviour belong together
  • you need many copies of the same shape
  • related types share behaviour
juggling parallel lists that travel together? that is a class asking to exist
