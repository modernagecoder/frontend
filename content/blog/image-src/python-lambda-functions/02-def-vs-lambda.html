Modern Age Codersdef vs lambda
the same function, two outfits
def a function with a life
def double(x):
    return x * 2
Named, reusable, can grow to many lines, carries a docstring.
lambda a function for a moment
lambda x: x * 2
Anonymous, one expression, born and used at the same spot.
double(5) == (lambda x: x * 2)(5) == 10
