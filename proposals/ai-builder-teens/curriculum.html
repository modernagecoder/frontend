The programme

What this is

A fast, hands-on two-month programme where teens go from knowing nothing about code to building and launching their own AI-powered apps. Every single class builds something real — no endless theory.

We start on the web (HTML, CSS and JavaScript) so students see results on screen from day one. Then we move into Python, the language of AI. Finally we go deep into the exciting part: machine learning and building apps with LLMs — chatbots, AI tools, "chat with your own notes", and a full-stack AI product — connecting everything using API keys and real AI models.

Three phases, one journey. Phase 1 · Web & JavaScript (Weeks 1–3) → build and deploy interactive apps · Phase 2 · Python (Weeks 4–5) → real programming and automation · Phase 3 · AI, ML & LLM apps (Weeks 6–8) → build and ship your own AI product.
Who it's for

Made for teenage builders (13–18)

  • Total beginners — no coding experience needed at all
  • Curious teens who love tech, games, or AI and want to build, not just use
  • Students who want a real portfolio of projects before college
  • Future creators & founders with app or startup ideas
  • Scratch / Python graduates ready to level up to real apps and AI
  • Anyone 13–18 who wants to actually ship something cool

The curriculum · 32 classes

Week-by-week syllabus

Four live classes a week for eight weeks. Each class lists what you learn and what you build.

Phase 1 · Weeks 1–3

Web & JavaScript — see results from day one

Build and deploy interactive web apps while learning how code really works.
Week 1 Your first web pages — HTML & CSS
01 How the Web Works + Your First Page

You'll learnWhat happens when you open a website; HTML basics — headings, text, links and images; using a code editor.

You'll buildYour first live web page — an "about me".

02 Structure & Forms

You'll learnPage structure, lists, buttons, forms and inputs, and semantic HTML.

You'll buildA profile page with a working contact form.

03 Styling with CSS

You'll learnColors, fonts, spacing and the box model; classes and selectors.

You'll buildRestyle your page so it actually looks great.

04 Layouts That Look Pro

You'll learnFlexbox and grid; responsive design that works on phone and laptop.

You'll buildA clean landing page for an idea you love.

Week 2 JavaScript — make it interactive
05 JavaScript Basics

You'll learnVariables, text and numbers, operators; running JavaScript in the browser.

You'll buildA tip / score calculator.

06 Logic & Functions

You'll learnif/else, loops, and writing your own reusable functions.

You'll buildA number-guessing game.

07 The DOM — Bring Pages to Life

You'll learnSelecting elements, handling clicks and events, and changing the page live.

You'll buildAn interactive to-do list.

08 Project — Interactive Quiz App

You'll learnPutting it together — app state, events and updating the UI.

You'll buildA multi-question quiz app that scores you.

Week 3 Real apps — data, APIs & going live
09 Working with Data — Arrays & Objects

You'll learnStoring lists and structured data; looping over data to build a UI automatically.

You'll buildA card gallery generated from data.

10 Talking to the Internet — APIs & JSON

You'll learnWhat an API is; fetch; reading JSON responses.

You'll buildAn app that pulls live data from a public API.

11 Async JavaScript

You'll learnPromises, async/await, loading states and handling errors.

You'll buildA search app with live results.

12 Ship It — Git, GitHub & Deploy

You'll learnSaving your work with Git & GitHub, then deploying to the web.

You'll buildYour app live on a real URL you can share.

Phase 2 · Weeks 4–5

Python — the language of AI

Learn real programming and automation, and get ready for machine learning.
Week 4 Python foundations
13 Meet Python

You'll learnWhy Python; setup; variables, types, and input / output.

You'll buildAn interactive text-based mini-app.

14 Logic in Python

You'll learnConditionals, loops, and writing clean, readable code.

You'll buildA quiz or calculator in Python.

15 Functions & Modules

You'll learnReusable functions, importing modules, and organizing your code.

You'll buildA small toolkit of your own functions.

16 Data Structures

You'll learnLists, dictionaries and sets — and when to use each.

You'll buildA command-line contact book or inventory.

Week 5 Python for real work
17 Files & Real Data (JSON)

You'll learnReading and writing files; working with JSON data that persists.

You'll buildA note-saver or expense tracker that remembers.

18 Libraries & Environments

You'll learnpip, installing packages, and virtual environments.

You'll buildSomething fun with a library — a QR code or an edited image.

19 Python Meets APIs

You'll learnThe requests library; calling APIs and handling responses in Python.

You'll buildA script that fetches and shows live data.

20 Project — Python Automation

You'll learnAutomating a boring task end-to-end.

You'll buildA mini web-scraper or an auto-report generator.

Phase 3 · Weeks 6–8

AI, Machine Learning & LLM Apps — the exciting part

Understand how AI works, then build and ship your own AI-powered product.
Week 6 Into AI & machine learning
21 What AI & ML Really Are

You'll learnAI vs machine learning vs deep learning; how machines "learn" from data — with real examples, no hype.

You'll buildExplore a real dataset and spot patterns.

22 Data Superpowers — pandas

You'll learnLoading, cleaning and exploring data with pandas; making simple charts.

You'll buildAnalyze a real dataset and chart your findings.

23 Your First ML Model

You'll learnFeatures & labels, train / test split, and making predictions with scikit-learn.

You'll buildA model that predicts something real.

24 Project — Prediction App

You'll learnChecking if your model is any good, then using it on brand-new input.

You'll buildA small app that makes predictions from your model.

Week 7 Building with LLMs — the fun part
25 How LLMs Work + Your First API Key

You'll learnWhat large language models do; getting and safely protecting an API key.

You'll buildYour very first call to an AI model from code.

26 Build an AI Chatbot

You'll learnSending prompts, reading responses, and building a chat loop.

You'll buildYour own chatbot with its own personality.

27 Prompt Engineering & Structured Output

You'll learnWriting great prompts and getting clean JSON back to power a real app.

You'll buildAn AI tool — a study-helper or idea generator.

28 AI Superpowers — Images & Voice

You'll learnSending images to AI, generating images, and text-to-speech basics (multimodal AI).

You'll buildAn app that describes or creates images.

Week 8 Connect everything & ship your AI product
29 Chat With Your Own Data (RAG)

You'll learnEmbeddings and retrieval — letting AI answer questions from your own documents.

You'll buildA "chat with my notes / PDF" app.

30 AI Agents — Apps That Take Action

You'll learnConnecting tools and APIs so AI can do things, not just talk.

You'll buildA mini AI agent that uses a real tool.

31 Full-Stack AI App

You'll learnConnecting a web front-end to a Python + LLM backend into one app.

You'll buildA polished AI web app, end-to-end.

32 Capstone — Ship Your AI Product + Showcase

You'll learnPlanning, building, testing and deploying a real product you're proud of.

You'll buildYour own deployed AI app with a shareable URL, which you present — and earn your Modern Age Coders certificate.
