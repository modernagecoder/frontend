Instructions

To test mobile layout:

  1. Open browser developer tools (F12)
  2. Toggle device toolbar (Ctrl+Shift+M)
  3. Select different mobile device sizes
  4. Verify that:
    • Hero section stacks vertically and centers content
    • All grids display as single column on mobile (< 480px)
    • Grids show 2+ columns on larger screens (> 480px)
    • No horizontal scrolling occurs
    • Proper spacing is maintained
