Common Issues

  • Script not loading: Check if path /js/unified-mobile-nav.js is correct
  • Elements not found: Check if IDs 'mobileMenuBtn' and 'navMenu' exist
  • Double initialization: Check if script is loaded twice
  • CSS conflicts: Check if other CSS is hiding the menu
