## üñºÔ∏è LOGO USAGE GUIDELINES

### LOGO SPECIFICATIONS

**Primary Logo:**
- File location: `/images/logo.png`
- Minimum size: 120px width
- Clear space: 20px on all sides
- Background: Works on dark backgrounds

**Logo Variations:**
- **Full Logo:** Logo + "Modern Age Coders" text
- **Icon Only:** Just the logo symbol (for small spaces)
- **Horizontal:** Logo + text side by side
- **Vertical:** Logo above text (stacked)

**On Dark Backgrounds:**
- Logo: Full color (Purple `#A855F7`)
- Text: White `#F8FAFC`

**On Light Backgrounds:**
- Logo: Full color (Purple `#A855F7`)
- Text: Dark `#0F172A`

**Monochrome Versions:**
- All White: For dark backgrounds
- All Black: For light backgrounds
- All Purple: For neutral backgrounds

### LOGO DON'TS
‚ùå Don't stretch or distort the logo  
‚ùå Don't change logo colors (except approved variations)  
‚ùå Don't add effects (shadows, glows, outlines)  
‚ùå Don't place on busy backgrounds without overlay  
‚ùå Don't rotate the logo

## üé® DESIGN ELEMENTS & PATTERNS

### GLASSMORPHISM EFFECT (Signature Style)

**Card Style:**
- **Background:** `rgba(16, 16, 28, 0.8)` - Semi-transparent dark
- **Border:** `1px solid rgba(255, 255, 255, 0.12)` - Subtle white
- **Blur:** 20px backdrop blur (if supported in Canva)
- **Shadow:** `0 8px 32px rgba(0, 0, 0, 0.3)`
- **Border Radius:** 24px (large curves)

**How to Create in Canva:**
1. Create rectangle with dark color `#10101C`
2. Reduce opacity to 80%
3. Add white border, 1px, 12% opacity
4. Add shadow: 8px blur, 32px spread, black 30%
5. Round corners to 24px

---

## ‚úÖ DO'S AND ‚ùå DON'TS

### COLOR DO'S ‚úÖ
‚úÖ Use dark backgrounds (`#000002`) as primary  
‚úÖ Use purple (`#A855F7`) as main brand color  
‚úÖ Apply glow effects to create depth  
‚úÖ Use gradients for buttons and CTAs  
‚úÖ Maintain high contrast for readability  
‚úÖ Use category colors consistently  

### COLOR DON'TS ‚ùå
‚ùå Don't use bright backgrounds for main content  
‚ùå Don't mix too many colors in one design  
‚ùå Don't use low contrast text  
‚ùå Don't change brand purple to other purples  
‚ùå Don't use gradients on body text  

---

### TYPOGRAPHY DO'S ‚úÖ
‚úÖ Use Inter font family consistently  
‚úÖ Use bold weights (700-800) for headings  
‚úÖ Maintain proper hierarchy (H1 > H2 > H3)  
‚úÖ Use white/light text on dark backgrounds  
‚úÖ Keep line height at 1.5-1.6 for readability  
‚úÖ Use proper letter spacing for headings  

### TYPOGRAPHY DON'TS ‚ùå
‚ùå Don't use more than 3 font weights per design  
‚ùå Don't use decorative fonts for body text  
‚ùå Don't use all caps for long text  
‚ùå Don't use font sizes below 14px  
‚ùå Don't mix multiple font families  

---

### DESIGN DO'S ‚úÖ
‚úÖ Use glassmorphism for cards and overlays  
‚úÖ Apply consistent border radius (24px for cards)  
‚úÖ Add glow effects to important elements  
‚úÖ Use proper spacing (16px minimum)  
‚úÖ Keep designs clean and minimal  
‚úÖ Use high-quality images  

### DESIGN DON'TS ‚ùå
‚ùå Don't overcrowd designs with too many elements  
‚ùå Don't use pixelated or low-quality images  
‚ùå Don't ignore spacing guidelines  
‚ùå Don't use conflicting visual styles  
‚ùå Don't add unnecessary decorations
