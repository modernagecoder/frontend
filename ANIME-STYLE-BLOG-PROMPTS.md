# üé® Style Guidelines for AI Generation

## Key Elements for Success:

### ‚úÖ DO Include:
- **Simple shapes and icons** instead of detailed screens
- **Anime/cartoon character style** with clean lines
- **Magical effects** (sparkles, glows, particles)
- **Purple (#8A2BE2) and teal (#4ECDC4)** color scheme
- **Gradient backgrounds** (purple to teal)
- **Floating elements** (blocks, symbols, icons)
- **Expressive characters** with simple features
- **Soft cel shading** and clean outlines

### ‚ùå AVOID:
- Detailed code on screens
- Realistic photography style
- Complex text or typography
- Cluttered compositions
- Realistic human faces
- Detailed computer interfaces
- Small intricate details

## Recommended AI Tools Settings:

### For Google Imagen:
```
Style: anime illustration, cartoon style, clean digital art
Quality: high
Avoid: realistic, photographic, detailed text
```

### For Midjourney:
```
Add to prompt: --style anime --niji 5 --ar 1.91:1 (for hero) or --ar 16:9 (for in-article)
```

### For DALL-E 3:
```
Paste prompt as-is, it will understand the anime style
```
